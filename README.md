# FreeBus (Demo)

A simple **educational Android app** that simulates a bus ticket SMS/message.  
This project is created **for learning and research purposes only**.  
Do **not** use it to forge, deceive, or impersonate any real transport system.

## Overview
  * 'app/src/main/java/...' — main source code.
  * 'app/src/main/res/...' — resources (layouts, drawables, values).
  * 'app/src/test' and 'app/src/androidTest' — unit and instrumentation tests.
  * 'gradle/', 'gradlew', 'build.gradle', 'settings.gradle' — build scripts.
  * Documentation: 'README.md', 'LICENSE', 'DISCLAIMER.md', '.gitignore'.

## Build and Run
1. Install **Android SDK** and **JDK**.  
2. Open the project in **Android Studio**, or build via terminal:
   - bash
   - ./gradlew assembleDebug
