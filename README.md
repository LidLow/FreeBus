# FreeBus (Demo)

A simple **educational Android app** that simulates a bus ticket SMS/message.  
This project is created **for learning and research purposes only**.  
Do **not** use it to forge, deceive, or impersonate any real transport system.

## Overview
  * 'app/src/main/java/...' — main source code.
  * 'app/src/main/res/...' — resources (layouts, drawables, values).
  * 'app/src/test' and 'app/src/androidTest' — unit and instrumentation tests.
  * 'gradle/', 'gradlew', 'build.gradle', 'settings.gradle' — build scripts.
  * Documentation: 'README.md', 'LICENSE', 'DISCLAIMER.md', '.gitignore'.

## Build and Run
1. Install **Android SDK** and **JDK**.  
2. Open the project in **Android Studio**, or build via terminal:
   - bash
   - ./gradlew assembleDebug
3. Install the demo APK on a test device or emulator:
   - adb install -r app/build/outputs/apk/debug/app-debug.apk

## Configuration
  * 'local.properties' is excluded from the repository. Create it locally if needed (path to Android SDK).
  * All example data are fictional. Never add real phone numbers, ticket data, API keys, or keystore files.

## Allowed Use
  * Study and explore Android development.
  * Build and run locally for educational demonstration.
  * Fork and modify for testing or research purposes.

## Forbidden Use
  * Creating or distributing fake or deceptive tickets.
  * Using any real transport operator names, data, or branding.
  * Publishing real credentials, keystores, or private data.
